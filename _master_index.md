### Adding New Topics

New knowledge topics are created only with explicit user approval. When approved, a new directory under `wiki/` will be created with:
- `wiki/<topic>/_index.md` — Local catalog for this topic
- `wiki/<topic>/_log.md` — Local operation log for this topic

## About This Wiki

This knowledge base uses a **graph-based structure** where pages connect to each other via `[[wikilinks]]`. Directories represent individual knowledge concepts, not broad categories.

**Philosophy**: Knowledge is not a hierarchy — it's a network. The wiki keeps growing richer as connections compound.
