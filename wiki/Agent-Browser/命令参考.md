# 命令参考

agent-browser 50+ 命令分类参考，隶属 [[Agent-Browser]] 工具。

---

## 导航操作

| 命令 | 说明 |
|------|------|
| `open <url>` | 打开网页 |
| `back` | 后退 |
| `forward` | 前进 |
| `reload` | 刷新 |

## 元素交互

| 命令 | 说明 |
|------|------|
| `click <selector>` | 点击元素（如 `@e2`） |
| `fill <sel> <text>` | 填充输入框（清空后填入） |
| `type <sel> <text>` | 逐字输入（模拟键盘） |
| `press <key>` | 按键（Enter、Tab 等） |
| `scroll <direction>` | 滚动页面（up/down） |

## 信息获取

| 命令 | 说明 |
|------|------|
| `get text <selector>` | 获取元素文本 |
| `get url` | 获取当前 URL |
| `is visible <selector>` | 检查元素是否可见 |

## 快照与截图

| 命令 | 说明 |
|------|------|
| `snapshot` | 生成完整可访问性树快照 |
| `snapshot -i` | 仅可交互元素快照 |
| `screenshot <file>` | 截图保存为文件 |

## 会话管理

| 命令 | 说明 |
|------|------|
| `--session <name> open <url>` | 创建命名会话 |
| `close` | 关闭当前会话 |

## 输出格式

| 参数 | 说明 |
|------|------|
| `--json` | 输出结构化 JSON |
| `--headed` | 可视化模式运行（调试用） |
