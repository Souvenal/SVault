# 框架类

UE 提供的基础框架类，用于扩展引擎功能。

---

## UBlueprintFunctionLibrary

继承此类后，类中的 `static` + `BlueprintCallable` 函数可作为工具函数在蓝图中直接调用。

```cpp
UCLASS()
class UMyFunctionLibrary : public UBlueprintFunctionLibrary
{
    UFUNCTION(BlueprintCallable, Category = "MyTools")
    static void MyUtilityFunction();
};
```
