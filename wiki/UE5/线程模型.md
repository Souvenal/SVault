# 线程模型

UE 采用双线程架构：Game Thread 与 Render Thread 并行执行，通过命令队列通信。

---

## 线程分工

| 线程 | 用途 | 入口函数 |
|------|------|----------|
| Game Thread | 游戏逻辑：Actor 移动、输入、蓝图执行 | `AActor::Tick()` |
| Render Thread | 渲染管线：RDG 调度、Shader 执行、RHICmdList | `BeginRenderViewFamily_RenderThread()` |

Game Thread 和 Render Thread 通过 `FRDGBuilder` 命令队列通信。

---

## `_RenderThread` 函数后缀

函数签名带此后缀，强制要求在 Render Thread 调用。约束：

- 只能访问 Render Thread 独占的 GPU 资源，如 `FRDGTextureRef`
- 禁止访问 Game Thread 对象，如 `UObject` 的 `UPROPERTY`
- 若参数带 `FRHICommandListImmediate&`，可直接提交 GPU 命令
