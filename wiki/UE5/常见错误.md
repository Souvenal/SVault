# 常见错误

UE5 开发中遇到的错误及解决方案。

---

## VisualStudioTools 模块找不到

```
Expecting to find a type to be declared in a module rules named 'VisualStudioTools' 
in 'UE5Rules, Version=0.0.0.0, Culture=neutral, PublicKeyToken=null'. 
This type must derive from the 'ModuleRules' type defined by UnrealBuildTool.
```

这是 Visual Studio Integration Tool 的配置问题。该工具为引擎开发准备，游戏项目不需要。

**解决方案（任选其一）：**

1. 在 VS 的 Unreal Engine 集成配置页面移除 Visual Studio Integration Tool
2. 在 `.uproject` 中移除 `VisualStudioTools` 的配置
