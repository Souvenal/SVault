# 会话管理

## BW_SESSION

`bw unlock` 成功后生成的临时访问令牌。

### 生成

```shell
bw unlock --raw
# 输出：eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 使用

**环境变量：**

```shell
export BW_SESSION=$(bw unlock --raw)
bw list items  # 无需再次输入主密码
```

**每次传递：**

```shell
bw list items --session $(bw unlock --raw)
```

---

## 生命周期

```
bw unlock → 生成 BW_SESSION → 会话开始
    ├── bw list items ✓
    ├── bw get password <id> ✓
    └── bw create item ... ✓
bw lock / 退出终端 / 超时 → BW_SESSION 失效
```

---

## 安全

`BW_SESSION` 等同于主密码权限：可读取所有密码、创建/修改/删除条目。

**不要持久化：**

```shell
# ❌ 错误
echo "BW_SESSION=xxx" >> ~/.bashrc

# ✓ 正确
export BW_SESSION=$(bw unlock --raw)
```

**用完锁定：**

```shell
bw lock  # 清除内存中的解密数据
```
