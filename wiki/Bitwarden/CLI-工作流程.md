# CLI 工作流程

## 三层架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   云端服务器     │     │   本地电脑      │     │   内存中（解密）  │
│  (加密数据)      │ ←── │  (加密缓存)     │ ←── │  (可读取密码)    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
       ↑                      ↑                      ↑
       │                      │                      │
    注册账号               bw login               bw unlock
    存储密码            （下载加密数据）           （主密码解密）
                                               ↓
                                        BW_SESSION
                                        临时访问令牌
```

云端只存密文，本地只缓存密文，内存中才解密。

---

## 命令

### bw login

```shell
bw login  # 下载加密数据到本地
```

连接云端，用账号密码验证身份，下载加密的密码库到本地缓存。数据仍是加密状态。

### bw unlock

```shell
bw unlock --raw  # 输入主密码解密，获取 BW_SESSION
```

`--raw` 直接输出令牌字符串，方便写入环境变量或脚本。

---

## 安全边界

| 层级 | 数据状态 | 风险 | 访问方式 |
|------|----------|------|----------|
| 云端服务器 | 加密 | 低 | 账号密码 |
| 本地电脑 | 加密 | 低 | 文件系统 |
| 内存 | 解密 | 高 | BW_SESSION |

主密码从未存储，只在 `bw unlock` 时输入。`BW_SESSION` 只在当前会话有效，退出后失效。即使本地缓存被盗，没有主密码也无法解密。
