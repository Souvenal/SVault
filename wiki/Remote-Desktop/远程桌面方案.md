# 远程桌面方案

远程控制电脑的两种方案：DIY 自建和商业软件。

---

## DIY 方案

**架构：** Sunshine（服务端） + Moonlight（客户端） + ZeroTier（虚拟局域网）

| 组件 | 角色 |
|------|------|
| Sunshine | 服务端，运行在被控机器上 |
| Moonlight | 客户端，运行在控制端 |
| ZeroTier | 创建虚拟局域网，穿透 NAT |

ZeroTier 管理面板：central.zerotier.com

---

## UU 远程

商业远程桌面软件，支持远程开机。

### 远程开机配置

网卡驱动设置：
- 启用「允许唤醒」
- 启用「Wake on Magic Packet」
