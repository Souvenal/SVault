## 析构函数中的移动语义

如果类的析构函数涉及指针释放，需要手写移动构造和移动赋值运算符。

### 移动构造

使用 `std::exchange` 实现：

```cpp
class Widget {
public:
    Widget(Widget&& other) noexcept 
        : ptr_(std::exchange(other.ptr_, nullptr)) {}
    
private:
    Resource* ptr_;
};
```

### 移动赋值

使用 `std::swap` 实现：

```cpp
class Widget {
public:
    Widget& operator=(Widget&& other) noexcept {
        std::swap(ptr_, other.ptr_);
        return *this;
    }
    
private:
    Resource* ptr_;
};
```
