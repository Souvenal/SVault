# CUDA 兼容性

RTX 5060 Ti GPU 的 CUDA 计算能力与 PyTorch 版本对应关系。

---

## 核心信息

| 项目 | 值 |
|------|-----|
| GPU | RTX 5060 Ti |
| 架构 | Blackwell |
| sm 计算能力 | sm_120 |
| 最低 CUDA 版本 | 12.8 |
| 对应 PyTorch 版本 | 2.7.1 |

## 说明

 Blackwell 架构（sm_120）需要 CUDA 12.8 及以上版本才能支持。PyTorch 2.7.1 是首个支持该配置的主流版本。
