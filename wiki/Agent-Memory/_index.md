## About

Agent Memory 是 AI 编程工具中跨会话持久化用户偏好、项目规范和工作流指令的能力。不同工具的实现方式差异较大：从文件系统级配置（CLAUDE.md / AGENTS.md / CODEBUDDY.md）到 GUI 面板设置，从手动编写到 AI 自动记忆，本主题梳理和对比这些实现。
