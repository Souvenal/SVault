# 动态链接库

Linux 动态链接器/加载器管理工具 `ldconfig`。

## ldconfig 作用

`ldconfig` 是动态链接库管理工具，用于配置和管理系统共享库：

- **更新动态链接库缓存**
- **创建符号链接**
- **维护库文件搜索路径**

## 工作原理

Linux 运行程序时需要查找共享库（.so 文件）。系统维护缓存文件 `/etc/ld.so.cache`，`ldconfig` 负责更新这个缓存。

常用命令：

```bash
# 更新缓存
sudo ldconfig

# 查看缓存内容
ldconfig -p

# 查看某目录的库
ldconfig -v /usr/local/lib
```
