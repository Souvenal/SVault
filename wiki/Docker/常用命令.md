# Docker 常用命令

镜像和容器操作的常用命令。

---

## docker pull

拉取镜像。

```bash
# 拉取官方镜像时直接使用镜像名，用户镜像需要添加用户名
docker pull username/repository:tag

# 不指定标签或 SHA256 时默认 latest
docker pull nginx[:tag|:SHA256]

# 拉取镜像的所有标签
docker pull [--all-tags|-a] ubuntu

# 拉取指定平台的镜像
docker pull --platform=linux/arm64 ubuntu
```

## docker images

查看本地镜像列表。

```bash
docker images
```

## docker build

构建镜像。

```bash
docker build -f <docker-file> -t <tag-name> <build-context-dir>
```

## docker run

运行容器。

```bash
# 挂载本地目录或文件到容器内部
docker run [--volume|-v] /host/path:/container/path some-image
```

## docker compose

多容器编排。

```yml
# docker-compose.yml
services:
  mcsema:
    image: some-image:latest
    platform: linux/amd64
    volumes:
      - .:/some-image/local
    entrypoint: /bin/bash
    tty: true        # 若不分配 tty，bash 会直接退出
    stdin_open: true
```

## docker exec

attach 到正在运行的容器。
