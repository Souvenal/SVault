# Android Gradle

Android 项目构建工具。

---

## 概述

Gradle 是 Android 的标准构建系统，基于 Groovy/Kotlin DSL 描述构建逻辑。

## 基础概念

- **Build Script**：构建脚本（`build.gradle`）
- **Task**：构建任务
- **Plugin**：扩展构建功能（如 Android 插件）

## 常用命令

```bash
./gradlew assembleDebug    # 编译 Debug 版本
./gradlew assembleRelease  # 编译 Release 版本
./gradlew clean            # 清理构建产物
```
